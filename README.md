LLVisu
======

LiberLocus Visualization Library

The purpose of this library is to support output format for state-of-the-art visualization softwares in terms of VTK, HDF5, XMF, Silo etc..

%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

To compile liberOut

$ cd LLVisu/liber
$ mkdir build
$ cd build
$ cmake ..
